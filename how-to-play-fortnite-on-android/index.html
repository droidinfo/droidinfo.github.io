Download the game from the Google Play Store


Before you can play Fortnite on Android, you need to download the game from the Google Play Store. This is a straightforward process, but there are a few things you need to know.

  • Make sure your device is compatible

    Not all Android devices are compatible with Fortnite. You can check the list of compatible devices on the Epic Games website.

  • You need a Google account

    To download Fortnite from the Google Play Store, you need to have a Google account. If you don't have one, you can create one for free.

  • The game is free to download

    Fortnite is a free-to-play game. However, there are some in-game purchases available.

  • The game is large

    The Fortnite download is large, so it may take some time to download. Make sure you have a strong internet connection before you start downloading.

Once you have downloaded Fortnite, you can create an Epic Games account and start playing.

Learn the controls


Mastering the controls is essential for playing Fortnite on Android. The game has a variety of controls that you need to learn in order to move around, shoot, and build.

  • Movement

    The left analog stick is used to move your character around. You can move forward, backward, left, and right. You can also sprint by pressing the left analog stick down.

  • Shooting

    The right analog stick is used to aim your weapon. You can shoot by pressing the right trigger. You can also use the left trigger to zoom in.

  • Building

    The four face buttons are used to build structures. The X button is used to build walls, the O button is used to build floors, the square button is used to build stairs, and the triangle button is used to build roofs.

  • Other controls

    There are a number of other controls that you need to know in order to play Fortnite on Android. These controls include the following:

    • The D-pad is used to select weapons and items.
    • The start button is used to open the menu.
    • The select button is used to open the map.

Once you have learned the controls, you can start practicing your skills in the game's Playground mode. This mode allows you to practice building and shooting without having to worry about other players.

Start playing!
